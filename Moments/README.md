# Moments SwiftUI

This is a re-implementation of [Moments App](https://github.com/lagoueduCol/iOS-linyongjian) using Swift 5.5 and SwiftUI. 

Features:
  * Aysnc/Await
  * Actor
  * AysncImage
  * MVVM
  * BFF
